# Music App (UI)

Music App is a UI-focused Android project built using **Kotlin** and **Jetpack Compose**. This project demonstrates the design and implementation of a modern music application interface with multiple screens, each representing a core music-related functionality. The project focuses purely on the **UI layer**, without backend integration or audio playback logic.
